PyPRPImporter
=============

The PyPRP Importer is a plugin for importing Plasma ages into Blender.


Requirements
------------

For installing on Windows you will need:

- Blender 2.49b 
- Python 2.6.6 
- Python Imaging Library 1.1.6 
- PyYAML 3.09



For installing on Mac/Linux you will need:
Mac/Linux

- Blender 2.49b 
- Python 2.6.6 
- Python Imaging Library 
- PyYAML 3.09
